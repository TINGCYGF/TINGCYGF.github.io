浏览器执行JS过程

浏览器分为渲染引擎和JS引擎

渲染引擎：解析HTML与CSS，俗称内核，如chrome的blink

JS引擎：读取网页在JavaScript代码并运行，如chrome的V8

JavaScript的组成

  1. ECMAScript（基础语法）：由ECMA国际标准化的一门语言。
  2. DOM（文档对象模型）
  3. BOM（浏览器对象模型）

变量

变量是存放数据的容器，通过变量名获取数据

本质：变量是程序内申请存放数据的空间

  • 声明多个变量：
  • 声明变量特殊情况
只声明不赋值 undefind
不声明不赋值 报错
不声明只赋值 正常输出（全局变量）
  • 变量命名规范
    • 由字母、数组、下划线、$符号组成
    • 区分大小写
    • 不能以数字开头
    • 不能是关键字：var、for
    • 变量名必须有意义
    • 遵守驼峰命名法。首字母小写，后单词大写：myFirstName

数据类型

  • 简单数据类型
Number 数字类型：包括整型和浮点型 0
Boolean 布尔值：true，false false
String 字符串类型 “”
Undefined 声明后没赋值 undefined
Null 空值 null

  • 数字类型中

    • 数字前面加0代表八进制
    • 数字前面加0x代表十六进制
    • 最大值：Number.MAX_VALUE
    • 最小值：Number.MIN_VALUE
    • lnfinity：代表无穷大
    • -lnfinity：无穷小
    • NaN：非数值
    • isNan() :用来判断是否为数值，返回true / false

  • 字符串类型

    • 字符串嵌套，外单内双 “ 我 ’ 是 ‘ 谁 ”
    • 字符串转义符：

    转义符都是 \ 开头 ， 并且都在引号中

\n 换行 newline
\\ \
\‘ \“ ’ “
\t tab缩进
\b 空格 blank
