Vue与axios网络请求封装方法

发表于 2019-03-23 更新于 2020-04-25 分类于 axios

Vue的作者尤大在Vue2的时候开始推荐axios作为网络请求工具，不再维护Vue旧有的工具。

axios是基于promise用于浏览器与node.js的http服务端的一种js。相当于连接桥，此桥是人工智能的，可以自定义浏览器与服务端请求与响应的方式。

axios主要特点：

  1. 支持浏览器与node.js

  2. 支持promise API

  3. 能够拦截请求和响应

  4. 能转换请求与响应数据

  5. 能取消请求

  6. 自动转换Json格式

  7. 浏览器端支持CSRF(跨域请求)
