编辑代码效率UP技巧

发表于 2019-04-07 更新于 2020-11-17 分类于 效率UP技巧

本文总结了一些Emmet语法使用技巧和编辑器快捷键以及插件，达到效率提高的作用

Emmet语法

Emmet语法注意点：中间不要留空格

!和html:5

​ ! 和 html:5 可以快速生成完整结构的html5代码

（子代）和 +（兄弟）

*（多个）和^（上一级）

()（分组）

属性

id属性、class属性、普通属性

$（数字顺序）

{}（内容）

隐式标签

自动填标签，若没下一级的就相同

如：div

若有下一级就自动加上下一级标签

如：ul、table

VS Code

配置

  • 自动保存 Auto Save: onFocusChange
  • 自动格式化 Format On Save 适合新人

快捷键

VS Code快捷键：

  • tab+shift

    • 减小选中行的缩进

  • ctrl + .

    • 让输入法在中英文标点符号之间来回进行切换

  • ctrl + /

  • 在代码当中快速打出注释

  • alt + shift + f

    • 代码格式化

一次选中多个内容:

  • alt + 鼠标点击
  • alt + shift + 鼠标拖动

  2. Terminal

    • 打开终端
    • 右键第一个即可打开

  3. Code Spell Checker

  4. Git Easy
