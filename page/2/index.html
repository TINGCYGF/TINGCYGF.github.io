运用typora编辑Markdown博客

发表于 2020-01-07 更新于 2020-05-25 分类于 Markdown

Typora是一款博主强烈推荐免费的Markdown编辑器，虽然免费但是使用起来非常简洁舒服。编辑博客和导出源码都非常方便。本博客就总结了运用Typora编辑博客常用的语法与快捷键。

PS：点击左下角的小圆圈即可调出大纲

Markdown语法与Typora中的快捷键

1. 一级到六级标题

一级标题语法：#号 + 空格

N级标题：N个#号 + 空格

快捷键：Ctrl + 1~6

取消标题：Ctrl + 0

2. 有序列表

有序列表语法： 数字1+ . +空格

行末回车自动添加。

行末按 “下” 则不会添加

快捷键： Ctrl + Shift + [

降级快捷键： Tab

升级快捷键：Shift + Tab

3. 无序列表

无序列表语法:

*+空格+内容

-+空格+内容

快捷键：Ctrl + Shift + ]

4. 任务列表

语法：“-” + “ [” + “空格” “] “ + “空格”

没有快捷键，可以点击界面：

  • 点击上面段落（O)—点击任务列表

5.插入表格

创建方式：|表头1|表头2|表头3|+回车

添加新一行：Ctrl+回车

快捷键：Ctrl + T 输入行：值 列：值

6.行内代码

创建方式：反引号+行内代码+反引号

快捷键：Ctrl + Shift + `

7.代码块

先进行相关设置：文件—>偏好设置—>勾选【显示行号】—>勾选【代码块自动换行】—>重启Typora

创建方式：``` + 代码块 + 三个反引号包围

快捷键：Ctrl + Shift + K

PS：在右下角输入语言名称即可自动更变颜色

8.插入图片

先进行相关设置：文件—>选择【复制图片到***文件夹】—>勾选【优先使用相对路径】—>重启Typora

创建方式：！—>选择图片

快捷键：Ctrl + Shift + i

最后还有很多功能没有总结出来，可以通过Typora菜单探索更多功能。如缩进功能、超链接功能等等，都是简单直白的。
